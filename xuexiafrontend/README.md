# xuexiafrontend

## 安装依赖

切换目录到 `xuexiafrontend`

```
npm install
```

### 运行项目
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
